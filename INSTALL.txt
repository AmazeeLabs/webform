Requirements
------------

Drupal CVS HEAD


Installation
------------

1. Create the SQL tables. This depends a little on your system, but the most
   common method is:
     mysql -u username -ppassword drupal < webform.mysql

2. Copy the webform.module to the Drupal modules/ directory.

3. Copy htmltags.inc under misc directory.
 
4. Enable the module in the admin interface.

5. To upgrade from a previous version go to "admin" -> "settings" -> 
   "webform" and select the upgrade link.
